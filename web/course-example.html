üìñ What is an Array?

An array is a collection of elements stored at contiguous memory locations. It's one of the simplest and most widely used data structures in programming.

Key Characteristics:

  • Fixed Size: Arrays have a predetermined size
  • Indexed Access: Elements can be accessed using indices (0-based)
  • Contiguous Memory: Elements are stored in adjacent memory locations
  • Same Data Type: All elements must be of the same type
// Example: Declaring and initializing an array
int[] numbers = {10, 20, 30, 40, 50};

// Accessing elements
int firstElement = numbers[0]; // 10
int lastElement = numbers[4]; // 50

// Modifying elements
numbers[2] = 35; // Array becomes {10, 20, 35, 40, 50}

‚è±Ô∏è Time Complexity

  • Access by Index: O(1) - Constant time
  • Search: O(n) - Linear time (unsorted array)
  • Insert at End: O(1) - If space available
  • Insert at Beginning: O(n) - Requires shifting
  • Delete: O(n) - Requires shifting

Let's explore some fundamental operations you'll frequently use with arrays:

1. Traversal

Visiting each element in the array sequentially.

for (int i = 0; i < array.length; i++) {
System.out.println(array[i]);
}

2. Searching

Finding an element in the array (Linear Search).

int linearSearch(int[] arr, int target) {
for (int i = 0; i < arr.length; i++) {
if (arr[i] == target) return i;
}
return -1; // Not found
}
